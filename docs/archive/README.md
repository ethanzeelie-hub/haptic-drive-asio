# Archive Notes

This folder keeps historical stage-by-stage material that is still useful for traceability, audits, and implementation archaeology, but should no longer be treated as the current product description.

Current-state docs live at the repository root and in `docs/adr/`:

- `ARCHITECTURE.md` for the live architecture,
- `KNOWN_ISSUES.md` for active issues only,
- `ROADMAP.md` for future work,
- `DEVELOPMENT_LOG.md` for chronological implementation history.
